# Review Rules

## قبل اعتماد أي تغيير

يجب التحقق من:

- هل يكرر شيئًا موجودًا؟
- هل يتبع معايير الشركة؟
- هل له ID إذا كان أصلًا معرفيًا؟
- هل تم تحديث Changelog؟
- هل يؤثر على مشاريع مستقبلية؟
- هل يستطيع AI فهمه بدون شرح خارجي؟

## قاعدة

أي تغيير كبير يجب أن يمر عبر Issue أو ADR.
