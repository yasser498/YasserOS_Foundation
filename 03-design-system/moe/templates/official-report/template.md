# MOE Official Report Template

## Page

- Size: A4
- Direction: RTL
- Orientation: Portrait by default
- Landscape allowed for wide tables

## Header

- MOE logo.
- School or department name.
- Report title.

## Content

- Executive summary.
- KPI cards.
- Data tables.
- Charts.
- Recommendations.

## Footer

- Page number.
- Date.
- Optional contact information.

## Signature

- Bottom right for Arabic official documents.
