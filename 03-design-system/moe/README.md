# MOE Design System

هذا المجلد يحول الهوية البصرية لوزارة التعليم إلى قواعد تنفيذية يمكن أن تستخدمها الأنظمة والتقارير والقوالب.

## مرجع الهوية
دليل الهوية البصرية لوزارة التعليم - أكتوبر 2024.

## عناصر أساسية من الدليل
- الهوية ليست شعارًا فقط؛ بل نظام متكامل من العناصر التصميمية والمبادئ التوجيهية.
- الشعار يتطلب مسافات آمنة حوله، وفي الطباعة يترك حد أدنى 1.5 سم عن أقرب حافة.
- يمنع تغيير ألوان الشعار أو نسبه أو استخدام ظلال خلفه أو وضعه على خلفيات مخالفة لقائمة الألوان.
- الخط العربي والإنجليزي المعتمد في الدليل: SAKKAL MAJALLA Regular / Bold.
