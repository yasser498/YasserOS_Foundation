# MOE Design Engine

محرك تصميم خاص بمخرجات وزارة التعليم داخل YasserOS.

## المكونات

- `tokens`: قيم التصميم القابلة لإعادة الاستخدام.
- `rules`: قواعد الاستخدام.
- `templates`: قوالب رسمية.
- `components`: مكونات تصميمية.
- `examples`: أمثلة مستقبلية.

## المبدأ

كل تصميم رسمي أو تعليمي يجب أن يعتمد على tokens وقواعد واضحة، وليس على اجتهاد بصري عشوائي.
