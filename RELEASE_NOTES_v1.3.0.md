# Release v1.3.0 - AI Engine

## الهدف

تأسيس أول طبقة ذكاء اصطناعي منظمة داخل YasserOS.

## معايير القبول

- لا يبدأ AI بالبرمجة قبل التحليل.
- كل مهمة AI لها مدخلات ومخرجات واضحة.
- كل مخرج يرتبط بمعيار أو Registry ID.
- دعم ربط Codex مستقبلاً.
- دعم OpenAI/Gemini/Claude/OpenRouter/Ollama نظريًا دون Vendor Lock-in.
- توثيق متى يستخدم النموذج القوي ومتى يستخدم نموذج أرخص أو محلي.
