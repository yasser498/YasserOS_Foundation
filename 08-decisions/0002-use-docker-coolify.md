# ADR 0002: استخدام Docker و Coolify

## القرار
استخدام Docker لتشغيل الخدمات، وCoolify لإدارة النشر من GitHub إلى VPS.

## السبب
- سهولة النشر.
- عزل المشاريع.
- قابلية النقل بين السيرفرات.
- مناسب لبناء منصة شركة وليس مشروعًا واحدًا.

## المخاطر
- يحتاج تنظيم جيد للموارد على VPS محدود.
- يجب ضبط النسخ الاحتياطي والمراقبة.
