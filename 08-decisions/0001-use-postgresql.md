# ADR 0001: استخدام PostgreSQL

## القرار
اعتماد PostgreSQL كقاعدة البيانات الأساسية.

## السبب
- قوية للأنظمة المؤسسية.
- تدعم العلاقات والقيود والفهارس.
- مناسبة للتقارير والتحليلات.
- سهلة النقل بين مزودي الاستضافة.

## البدائل
- MySQL
- Firebase
- MongoDB
- SQLite

## متى نراجع القرار؟
إذا ظهر احتياج تقني لا تخدمه PostgreSQL بكفاءة، أو احتجنا قاعدة متخصصة بجانبها مثل Redis أو ElasticSearch.
