# PostgreSQL Core Standard

## القاعدة العامة

PostgreSQL هي قاعدة البيانات الأساسية في مشاريع YasserOS.

## حقول قياسية لكل جدول

كل جدول تشغيلي يجب أن يحتوي على:

- `id UUID PRIMARY KEY`
- `created_at TIMESTAMPTZ`
- `updated_at TIMESTAMPTZ`
- `deleted_at TIMESTAMPTZ NULL`
- `created_by UUID NULL`
- `updated_by UUID NULL`
- `is_active BOOLEAN DEFAULT TRUE`

## قواعد التصميم

- لا تكرار للبيانات.
- العلاقات تتم عبر Foreign Keys.
- استخدام Migrations.
- استخدام Indexes للأعمدة المستخدمة في البحث.
- استخدام Soft Delete للجداول التشغيلية.
- توثيق كل جدول ووظيفته.
