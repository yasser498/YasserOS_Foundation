# معيار Backend

## الافتراضي
Node.js + TypeScript.

## الطبقات
- Controller: استقبال الطلب فقط.
- Service: منطق العمل.
- Repository: التعامل مع قاعدة البيانات.
- DTO/Validation: التحقق من المدخلات.
- Policy/Guard: الصلاحيات.

## المتطلبات الثابتة
- Logging.
- Error Handling.
- Rate Limiting.
- Audit Trail.
- API Documentation.
