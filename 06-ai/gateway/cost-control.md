# AI Cost Control

## قواعد ضبط التكلفة

- استخدم cache للطلبات المتكررة.
- لا ترسل ملفات كاملة دون حاجة.
- أرسل السياق اللازم فقط.
- استخدم نماذج اقتصادية للمهام البسيطة.
- استخدم النموذج القوي فقط للقرارات المعمارية أو التحليل العميق.
- خزّن نتائج التحليل المعتمدة في Company Memory.

## مؤشرات مراقبة

- تكلفة كل مشروع.
- تكلفة كل مستخدم.
- تكلفة كل نوع مهمة.
- عدد الطلبات الفاشلة.
