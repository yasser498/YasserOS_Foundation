# MOE Design Review Prompt

```text
راجع التصميم التالي وفق هوية وزارة التعليم.

تحقق من:
- سلامة استخدام الشعار.
- عدم تغيير النسب أو الألوان.
- وضوح RTL.
- قابلية الطباعة.
- وضوح الجداول.
- مناسبة الألوان.
- عدم ازدحام التصميم.
- مناسبة الخطوط.
- جودة المخرجات الرسمية.

أعطني:
1. درجة الالتزام من 100.
2. الأخطاء.
3. التحسينات.
4. قرار: مقبول / يحتاج تعديل / مرفوض.
```
