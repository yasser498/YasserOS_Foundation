# Company Memory Rules

## الفرق بين Knowledge وMemory

Knowledge = معلومة.

Memory = خبرة مع سبب وسياق ونتيجة.

## ما الذي يحفظ؟

- قرار ناجح.
- خطأ تكرر.
- قالب أثبت نجاحه.
- مكون أعيد استخدامه.
- برومبت ممتاز.
- مشكلة أمنية.
- تحسين أداء.

## صيغة الحفظ

كل Memory يجب أن تحتوي على:

- السياق.
- القرار.
- السبب.
- النتيجة.
- طريقة إعادة الاستخدام.
