# YasserOS AI Engine

محرك الذكاء الاصطناعي في YasserOS ليس مجرد برومبتات، بل طبقة تشغيل مسؤولة عن:

- التحليل.
- التخطيط.
- توليد الكود.
- مراجعة الكود.
- تصميم قواعد البيانات.
- توليد التقارير.
- مراجعة الهوية.
- توثيق القرارات.
- اقتراح التحسينات.
